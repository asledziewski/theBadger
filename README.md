# theBadger

Aplikacja wykonana na przedmiot Języki formalne i kompilatory na 5. semestrze studiów. Umożliwia ona przeglądanie zawartości plików jar (klas, atrybutów, konstruktorów i metod), oraz ich edytowanie np. poprzez dodawanie nowych klas, edycję metod itp.

## Wykorzystane frameworki

* [Maven](https://maven.apache.org/)
* [JavaFX](https://www.java.com/pl/)
* [JavaAssist](http://www.javassist.org/)
